Acorn

Acorn oli kitaransoiton itseopiskeluun tarkoitettu pelillistetty mobiilisovellus, joka toteutettiin osana Jyväskylän Ammattikorkeakoulun Ticorporate-opintojaksoa. Vastuullani projektissa oli Product Ownerin sekä full stack-ohjelmoijan tehtävät. Työskentelimme projektin parissa noin yhdeksän kuukautta, jonka aikana opin runsaasti sovelluskehityksen eri vaiheista ja sovellusten markkinoinnista.

Projekti kerrytti minulle seuraavaa osaamista:

  • Angular
  • Tensorflow
  • Express.js
  • PostgreSQL
  • Git
  • Agile-menetelmät
  • Tiimityöskentely

Github Classrooms -verkkokurssin toteuttaminen

Projektin tarkoituksena oli vähentää opettajan työtaakkaa tehtävien tarkastamisessa automatisoimalla Git-tehtävien tehtäväpalautukset Github Actionsin ja Github Classroom- ympäristön avulla. Projektissa sain tutustua Python-skriptaukseen, testilähtöiseen suunnitteluun
